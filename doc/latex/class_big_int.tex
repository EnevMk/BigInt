\doxysubsection{Detailed Description}
The main class for the whole project.

Class \mbox{\hyperlink{class_big_int}{Big\+Int}}, its purpose being enabling users to work with numbers too big that aren\textquotesingle{}t supported by the language built-\/in types. Working in a billion number system, it offers broad functionality along with great effectiveness resource-\/wise.

Core functionalities are\+: Addition, Subtraction, Multiplication and a fast exponentiation alghoritm.
